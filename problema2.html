Somar N valores até N < 0

Dicas para resolver possiveis erros do seu código

Dica 1 -> Ao apenas criar um while loop e inserir o bloco de instrução indicando que ele deve somar, ocorrerá um problema. O programa deve somar apenas os valores positivos, certo? Uma vez que valores negativos são apenas para finalizar o programa. Acontece que ele efetuará o cálculo de números negativos. Para resolver esse problema, foi necessário o uso de um if e dentro do bloco de instrução foi realizado o cálculo da soma. Sendo a condição: if n >= 0.
